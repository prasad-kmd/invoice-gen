# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-20

### Added
- Initial project structure for PC Repair Invoice Generator.
- Integration with existing Supabase and Better Auth backend.
- Admin-only access via role-based middleware.
- Dashboard with real-time financial stats and recent invoices.
- Client management with CRUD operations and history tracking.
- Dynamic Invoice creation and editing with real-time preview.
- Professional PDF generation and download via `@react-pdf/renderer`.
- Business settings and invoice formatting configuration.
- Tailwind CSS 4 styling and dark mode support.
- Comprehensive documentation and setup guides.
